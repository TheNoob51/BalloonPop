## üéÆ How to Play

1.  **Start:** Press the "Play" button from the main menu.
    
2.  **Pop:** Red balloons will float up from the bottom of the screen. Tap them to pop them!
    
3.  **Score:** Your score increases for every balloon you successfully pop.
    
4.  **Game Over:** The game ends if you miss a balloon (or after a set time/condition).
    
5.  **Replay:** From the "Game Over" screen, you can tap the **Retry** button to play again or the **Exit** button to return to the main menu.
